Discovery Project: Smart Environmental Monitoring Node

This project focused on monitoring temperature, humidity, and light conditions in shared lab spaces using an embedded microcontroller. The system recorded environmental changes in real time and visualized data for analysis and optimization.

Results

  • Designed a reliable sensor network prototype.
  • Collected accurate readings and visualized trends.
  • Documented setup and results for replication.
